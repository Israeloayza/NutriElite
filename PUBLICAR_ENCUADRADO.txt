NUTRIELITE + ENCUADRADO INTEGRADO

ARCHIVOS NUEVOS:
- agendar.html
- agendar-esencial-presencial.html
- agendar-esencial-online.html
- agendar-acompanamiento.html
- agendar-integral.html

FUNCIONAMIENTO:
- Los botones principales de la landing abren páginas internas.
- Cada página carga un único calendario de Encuadrado.
- El script de Encuadrado aparece una sola vez en cada página.
- Esencial abre primero un selector de modalidad.
- Acompañamiento e Integral abren directamente su calendario.
- El botón general de agenda abre el selector de programas.

PUBLICACIÓN:
1. Descomprime el ZIP.
2. En GitHub, Add file > Upload files.
3. Sube TODOS los archivos a la raíz del repositorio.
4. Conserva los iconos originales existentes.
5. Commit sugerido:
   Integra agenda de Encuadrado en NutriElite
6. Espera el despliegue en Actions.
7. Prueba los cuatro recorridos desde celular y computadora.

NO PUBLIQUES NI COMPARTAS CLAVES API.
Esta integración utiliza el widget oficial y no necesita API key.
