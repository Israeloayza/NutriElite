CONECTAR ENCUADRADO
1. Abre index.html.
2. Busca:
   const BOOKING_URL = "";
3. Pega tu enlace:
   const BOOKING_URL = "https://TU-ENLACE-DE-ENCUADRADO";
4. Guarda.

PUBLICAR EN GITHUB
1. Entra a https://github.com/Israeloayza/NutriElite
2. Abre index.html.
3. Pulsa el lápiz para editar.
4. Borra el contenido actual.
5. Pega el contenido del nuevo index.html.
6. En Commit changes escribe:
   Actualiza NutriElite con agenda y laboratorios
7. Pulsa Commit changes.
8. Repite con site.webmanifest si aparece modificado.
9. Desde Add file > Upload files sube robots.txt y sitemap.xml si aún no existen.
10. Abre https://israeloayza.github.io/NutriElite/ y prueba todo desde celular.
