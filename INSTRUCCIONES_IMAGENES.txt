CÓMO SUSTITUIRLAS
1. Descarga cada fotografía.
2. Renombra exactamente según se indica.
3. Conserva la extensión .jpg.
4. Sustituye los archivos provisionales dentro de assets/images.
5. Sube la carpeta completa a GitHub.

PUBLICAR EN GITHUB
1. Entra al repositorio NutriElite.
2. Reemplaza index.html por el nuevo.
3. Usa Add file > Upload files.
4. Arrastra la carpeta assets o sus archivos conservando la estructura:
   assets/images/
5. Confirma con Commit changes.
6. Abre la página en modo incógnito para verificar.

La página ya utiliza rutas locales y no depende de enlaces externos de Unsplash.
